# 🚀 RM Agency — High Conversion Landing Pages

Landing page oficial da **RM Agency**, desenvolvida com foco em performance, experiência do usuário e conversão.

## 💼 Sobre a RM Agency

A RM Agency é uma agência digital especializada na criação de **landing pages premium e experiências web modernas**, com foco em:

* Alta conversão
* Performance
* Design estratégico (UI/UX)
* Experiência imersiva estilo Apple/Stripe

## 🎯 O que este projeto entrega

* Interface moderna e responsiva
* Animações avançadas (scroll-based)
* Estrutura otimizada para conversão
* Integração com WhatsApp como canal de contato
* Experiência fluida e envolvente

## 🛠️ Tecnologias

* React
* TypeScript
* Tailwind CSS (v4)
* Framer Motion
* Vite

## ⚙️ Como rodar o projeto

```bash
npm install
npm run dev
```

## 📈 Objetivo

Este projeto foi desenvolvido como base para **entregas reais da RM Agency**, sendo utilizado como referência para criação de landing pages para clientes.

## 🧠 Diferenciais

* Design orientado à conversão
* Estrutura reutilizável para múltiplos clientes
* Código limpo e escalável
* Foco em performance e experiência do usuário
